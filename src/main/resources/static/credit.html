Project Context

This project was undertaken by first-year, second-semester students from the Faculty of Computing at the Sri Lanka Institute of Information Technology (SLIIT). It serves as a practical application of concepts learned in our Object-Oriented Programming and Data Structures and Algorithms modules.

This project was developed as part of our academic curriculum, applying concepts from our studies to create a functional web application.
